Los reclutadores técnicos dedican **menos de dos minutos** al primer vistazo. Tu trabajo es que en ese tiempo vean señal clara de seniority.

## GitHub que convence

- README en el repo estrella: problema, arquitectura, cómo correrlo, decisiones (por qué Next, por qué Postgres).
- Commits limpios o al menos ramas con PRs descriptivos.
- Evita repos vacíos o solo forks sin cambios.

## Portfolio web (una página basta)

Hero con rol + stack, 2–3 casos con **métricas**, enlace a código o demo, y forma de contacto. El SEO de tu nombre propio importa: usa título y meta description únicos.

## LinkedIn optimizado

Titular con especialidad y nicho (*Backend payments*, *Data eng en ads*). About con historia en tres párrafos: qué construyes, cómo lo haces, qué buscas ahora. Skills alineados con las vacantes que quieres.

En mentorías revisamos capturas y damos **feedback accionable** en una sola sesión.
